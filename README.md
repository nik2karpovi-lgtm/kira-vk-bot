# 🖤 Kira VK Bot

Бот для сообщества ВКонтакте @kira_musicant

## Что умеет

- Отвечает на сообщения от подписчиков
- Умные ответы на популярные вопросы
- Рассказывает о заказе треков
- Направляет на подборки Ozon

## Переменные окружения (Railway)

- `VK_TOKEN` — токен группы VK
- `CONFIRMATION_CODE` — код подтверждения VK
- `GROUP_ID` — ID группы VK
- `PORT` — порт (Railway ставит автоматически)

## Деплой на Railway

1. Загрузи код на GitHub
2. Подключи репозиторий к Railway
3. Добавь переменные окружения
4. Railway сам задеплоит

## Настройка VK

1. Управление группой → Работа с API
2. Создай ключ доступа (сообщения + стена)
3. Callback API → добавь URL от Railway
4. Скопируй код подтверждения
